# Polymat

Polymer material mechanics utilites for Python

Main Features:

- Material models library:
    - Hyperelastic
    - Viscoelastic
    - Viscoplastic 
- Computation of Strain, Stress quantities
- Simulation of main deformation modes
- Material parameter calibration from test data

The code implementations are heavily inspired by the examples in [1].

References
----------

[1] J. Bergstrom, Mechanics of Solid Polymers, Elsevier, 2015
